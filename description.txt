Таблицы моей базы данных:
1. Склады (`warehouses`); 
   Таблица для хранения информации о складах.  
   - `warehouse_id`: Уникальный идентификатор склада (первичный ключ).  
   - `warehouse_name`: Название склада (обязательное поле).  

2. Контрагенты (`contractors`): 
   Таблица для хранения информации о контрагентах (поставщики, покупатели).  
   - `contractor_id`: Уникальный идентификатор контрагента (первичный ключ).  
   - `contractor_name`: Название контрагента (обязательное поле).  

3. Категории товаров (`categories`):  
   Таблица для группировки товаров по категориям.  
   - `category_id`: Уникальный идентификатор категории (первичный ключ).  
   - `category_name`: Название категории (обязательное поле).  

4. Бренды товаров (`brands`):  
   Таблица для хранения информации о брендах товаров.  
   - `brand_id`: Уникальный идентификатор бренда (первичный ключ).  
   - `brand_name`: Название бренда (может быть NULL, если бренд не указан).  

5. Товары (`products`): 
   Таблица для хранения информации о товарах.  
   - `product_id`: Уникальный идентификатор товара (первичный ключ).  
   - `product_name`: Название товара (обязательное поле).  
   - `category_id`: Ссылка на категорию товара (внешний ключ, связывает с таблицей `categories`).  
   - `brand_id`: Ссылка на бренд товара (внешний ключ, связывает с таблицей `brands`).  

6. Поступления товаров (`deliveries`): 
   Таблица для учета поставок товаров на склады.  
   - `delivery_id`: Уникальный идентификатор поступления (первичный ключ).  
   - `product_id`: Ссылка на товар (внешний ключ, связывает с таблицей `products`).  
   - `warehouse_id`: Ссылка на склад (внешний ключ, связывает с таблицей `warehouses`).  
   - `contractor_id`: Ссылка на контрагента (внешний ключ, связывает с таблицей `contractors`).  
   - `quantity`: Количество поступившего товара (обязательное поле).  
   - `delivery_date`: Дата поступления товара (обязательное поле).  

7. Выдача товаров со склада (`withdrawals`): 
   Таблица для учета выдачи товаров со склада.  
   - `withdrawal_id`: Уникальный идентификатор выдачи (первичный ключ).  
   - `product_id`: Ссылка на товар (внешний ключ, связывает с таблицей `products`).  
   - `warehouse_id`: Ссылка на склад (внешний ключ, связывает с таблицей `warehouses`).  
   - `contractor_id`: Ссылка на контрагента (внешний ключ, связывает с таблицей `contractors`).  
   - `quantity`: Количество выданного товара (обязательное поле).  
   - `withdrawal_date`: Дата выдачи товара (обязательное поле).  

8. Перемещение товаров между складами (`transfers`): 
   Таблица для учета перемещений товаров между складами.  
   - `transfer_id`: Уникальный идентификатор перемещения (первичный ключ).  
   - `product_id`: Ссылка на товар (внешний ключ, связывает с таблицей `products`).  
   - `from_warehouse_id`: Ссылка на склад отправления (внешний ключ, связывает с таблицей `warehouses`).  
   - `to_warehouse_id`: Ссылка на склад получения (внешний ключ, связывает с таблицей `warehouses`).  
   - `quantity`: Количество перемещенного товара (обязательное поле).  
   - `transfer_date`: Дата перемещения товара (обязательное поле).
